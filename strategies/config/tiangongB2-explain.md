# 天宫B2策略通达信公式详解

本文档详细解读 `tiangongB2.txt` 中的通达信公式实现逻辑。

---

## 1. 概述

天宫B2策略是一个基于多维度技术指标综合评分的选股系统。它通过31个得分项对股票进行全方位的技术面评估，最终筛选出符合买入条件的股票。

### 核心设计思想
- **趋势判断**: 使用EMA、BBI判断多空趋势
- **动量指标**: KDJ、MACD、RSI判断超买超卖
- **量价配合**: 成交量与价格走势的配合分析
- **形态识别**: 识别关键K线形态（大长阳、大长阴、暴力K等）

---

## 2. 参数定义

```tcl
M1:=14;M2:=28;M3:=57;M4:=114;N1:=3;N2:=14;N3:=28;N4:=57;
```

| 参数 | 值 | 用途 |
|------|-----|------|
| M1 | 14 | 短期均线周期 |
| M2 | 28 | 中期均线周期 |
| M3 | 57 | 中长期均线周期 |
| M4 | 114 | 长期均线周期 |
| N1 | 3 | 短期RSI周期 |
| N2 | 14 | 中期RSI周期 |
| N3 | 28 | 长期RSI周期 |
| N4 | 57 | 超长RSI周期 |

## 3. 趋势线系统

### 3.1 知行短期趋势线

```tcl
知行短期趋势线:=EMA(EMA(C,10),10),COLORFFFFFF,LINETHICK1;
```

**实现逻辑**: 对收盘价进行双重EMA平滑
- 第一次EMA: `EMA(C,10)` - 10日指数移动平均
- 第二次EMA: `EMA(EMA(C,10),10)` - 对上述结果再进行一次10日EMA

**作用**: 消除短期噪音，提供更平滑的短期趋势线

### 3.2 知行多空线

```tcl
知行多空线:=(MA(CLOSE,M1)+MA(CLOSE,M2)+MA(CLOSE,M3)+MA(CLOSE,M4))/4;
```

**实现逻辑**: 4条不同周期均线的算术平均值
- M1=14日均线
- M2=28日均线
- M3=57日均线
- M4=114日均线

**作用**: 多空分界线，当价格位于多空线上方时偏多，下方时偏空

### 3.3 知行线波动平均

```tcl
知行线波动平均 := (REF(知行多空线,15) + REF(知行多空线,30) + REF(知行多空线,45) + REF(知行多空线,60)) / 4;
知行线平均_今 := IF(知行线波动平均 <> 0, (知行多空线 - 知行线波动平均)/知行线波动平均, 0);
```

**作用**: 计算当前多空线相对于过去15-60日均值的偏离程度，衡量趋势强度

## 4. 技术指标系统

### 4.1 KDJ随机指标

```tcl
RSV:=(C-LLV(L,9))/(HHV(H,9)-LLV(L,9))*100;
K:=SMA(RSV,3,1);
D:=SMA(K,3,1);
J:=3*K-2*D;
KDJ_J低 := J < 13;
```

| 指标 | 计算公式 | 说明 |
|------|----------|------|
| RSV | (收盘价-9日最低价)/(9日最高价-9日最低价)×100 | 原始随机值 |
| K | SMA(RSV,3,1) | 3日平滑的K值 |
| D | SMA(K,3,1) | 3日平滑的D值 |
| J | 3K-2D | 敏感的超买超卖指标 |

**信号解读**:
- J < 13: 超卖区域，可能出现反弹
- J > 80: 超买区域，可能出现回调

### 4.2 MACD指标

```tcl
DIF:=EMA(C,12)-EMA(C,26);
DEA:=EMA(DIF,9);
MACD_多头 := DIF >= 0;
```

| 指标 | 计算公式 | 说明 |
|------|----------|------|
| DIF | EMA(12) - EMA(26) | 快线 |
| DEA | EMA(DIF,9) | 慢线 |
| MACD柱 | (DIF-DEA)×2 | 柱状图 |

**信号解读**:
- DIF >= 0: 多头排列，看涨
- DIF < 0: 空头排列，看跌

### 4.3 RSI相对强弱指标

```tcl
LC:=REF(CLOSE,1);
RSI1:=SMA(MAX(CLOSE-LC,0),N1,1)/SMA(ABS(CLOSE-LC),N1,1)*100;
RSI2:=SMA(MAX(CLOSE-LC,0),N2,1)/SMA(ABS(CLOSE-LC),N2,1)*100;
RSI3:=SMA(MAX(CLOSE-LC,0),N3,1)/SMA(ABS(CLOSE-LC),N3,1)*100;
RSI4:=SMA(MAX(CLOSE-LC,0),N4,1)/SMA(ABS(CLOSE-LC),N4,1)*100;
枢轴线:=MA(RSI1,25);
```

| RSI周期 | 指标用途 |
|---------|----------|
| RSI1 (3日) | 极短期超买超卖 |
| RSI2 (14日) | 短期强弱 |
| RSI3 (28日) | 中期强弱 |
| RSI4 (57日) | 长期强弱 |
| 枢轴线 | RSI1的25日均值，作为多空参考线 |

### 4.4 BBI多空指数

```tcl
BBI:=(MA(C,3)+MA(C,6)+MA(C,12)+MA(C,24))/4;
前20日BBI:=REF(BBI,20);
```

**实现逻辑**: 4条不同周期均线的平均值
- 3日、6日、12日、24日均线的算术平均

**信号**: 价格 > BBI 为多头市场，价格 < BBI 为空头市场

### 4.5 其它价格指标

```tcl
涨幅:=(C-REF(C,1))/REF(C,1)*100;
振幅:=(H-L)/REF(C,1)*100;
K线长度:=HIGH-LOW;
上影线:=HIGH-MAX(C,O);
下影线:=MIN(C,O)-LOW;
实体 := O - C;
```

| 指标 | 说明 |
|------|------|
| 涨幅 | 当日涨跌幅 |
| 振幅 | 当日振幅 |
| K线长度 | 整根K线的长度 |
| 上影线 | 上影线长度 |
| 下影线 | 下影线长度 |
| 实体 | 实体部分（阴线为负） |

## 5. K线形态识别

### 5.1 大长阳/大长阴

```tcl
波幅:=MA(TR,30);
涨跌幅:=(C-REF(C,1))/REF(C,1)*100;
波动率:=波幅/REF(C,1)*100;
大长阳:=C>O AND 涨跌幅>波动率*1.5 AND 涨跌幅>2;
大长阴:=C<O AND ABS(涨跌幅)>波动率*1.1 AND ABS(涨跌幅)>2;
```

| 形态 | 条件 | 市场含义 |
|------|------|----------|
| 大长阳 | 收盘>开盘 且 涨幅>波动率×1.5 且 涨幅>2% | 强势上涨 |
| 大长阴 | 收盘<开盘 且 跌幅>波动率×1.1 且 跌幅>2% | 强势下跌 |

### 5.2 关键K线

```tcl
关键K:=CLOSE>REF(CLOSE,1) AND VOL>参考成交量*1.8 AND 大长阳 AND VOL>MA(VOL,40);
```

**关键K线条件** (全部满足):
1. 收盘价上涨
2. 成交量大于参考成交量×1.8倍
3. 符合大长阳形态
4. 成交量大于40日均量

### 5.3 暴力K线

```tcl
暴力K := C > REF(C, 1) AND VOL > 参考成交量 * 1.8 AND 涨跌幅 > 4 AND 上影线 <= K线长度 / 4 AND VOL > MA(VOL, 60)
    AND (REF(C, 1) - REF(O, 9)) / REF(O, 9) * 100 < 4 AND (REF(C, 1) - REF(O, 4)) / REF(O, 4) * 100 < 4;
```

**暴力K条件** (全部满足):
1. 收盘价上涨
2. 成交量大于参考成交量×1.8倍
3. 涨幅大于4%
4. 上影线长度不超过K线长度的1/4（几乎没有上影线）
5. 成交量大于60日均量
6. 前9日涨幅小于4%（排除前期大幅上涨）
7. 前4日涨幅小于4%（排除短期加速上涨）

### 5.4 次高点判断

```tcl
次高点基础:=HHV(H, 4) = HHV(H, 60) AND H <> HHV(H, 60)         						   	 
		    AND (VOL > MA(VOL, 5) OR VOL > MA(VOL, 10))             					 
    	    AND C < O        															  
    		    AND (((C / REF(C, 10) - 1) * 100 > 10) OR ((C / REF(C, 50) - 1) * 100 > 50)); 
次高点判断:=IF(次高点基础,
            IF(前3天最高点位置 >= 0,
            IF(VOL >= REF(VOL, 前3天最高点位置) * 0.80, 1,0),0),0);
```

识别近4日高点是60日内次高点的形态，表示可能形成双顶或多重顶

## 6. 成交量系统

### 6.1 参考成交量

```tcl
参考成交量:=IF(REF(VOL,1)<=VOL/8,REF(VOL,2),REF(VOL,1));
```

**逻辑**: 如果昨日成交量极小（<=今日成交量/8），则用前一日成交量作为参考，避免地量干扰

### 6.2 倍量柱/天量柱

```tcl
倍量柱:=IF(VOL>参考成交量*1.8,1,0) AND C>O AND C>REF(C,1);
天量柱:=REF(倍量柱, 1) AND VOL >= REF(VOL, 1) * 1.8;
```

| 形态 | 条件 | 含义 |
|------|------|------|
| 倍量柱 | 成交量>参考成交量×1.8 且 收阳线 且 收盘上涨 | 成交量明显放大 |
| 天量柱 | 前一日是倍量柱 且 今日成交量>=昨日×1.8 | 阶段性天量 |

### 6.3 成交量均线

```tcl
VOL_MA60:=MA(VOL,60);
```

60日均量线，用于判断成交量活跃度

## 7. 上证指数关联

```tcl
上证收盘:= "999999$CLOSE";       
上证昨高:= REF("999999$HIGH", 1);   
上证30日高:= HHV(REF(上证收盘, 1), 29);
```

| 指标 | 说明 |
|------|------|
| 上证收盘 | 上证指数收盘价 |
| 上证昨高 | 上证指数昨日最高价 |
| 上证30日高 | 上证指数30日最高价 |

**作用**: 同步参考大盘走势，避免逆大盘而动

## 8. B2触发条件

```tcl
B2触发:=((REF(J,1)<=21) OR (REF(RSI1,1)<=21)) AND VOL>REF(VOL,1) AND 涨幅>3.95 AND VOL > VOL_MA60 AND C>O;
```

**B2触发条件** (全部满足):
1. 昨日J值<=21 或 昨日RSI1<=21 (超卖状态)
2. 今日成交量大于昨日成交量 (放量)
3. 涨幅>3.95% (明显上涨)
4. 成交量大于60日均量 (活跃)
5. 收阳线 (收盘上涨)

**解读**: 捕捉超卖反弹的启动点

## 9. 得分系统详解

### 9.1 动量得分

#### 得分1: J值得分
```tcl
得分1:=IF(J>80,-3,0) + IF(J>65,-0.8,0) + IF(J>55,-0.8,0);
```
- J>80: -3分 (超买风险)
- J>65: -0.8分
- J>55: -0.8分

#### 得分2: MACD多头得分
```tcl
得分2:=IF(DIF>=0,0.3,0); 
```
- DIF>=0: +0.3分 (多头趋势)

#### 得分3: 均线多头排列得分
```tcl
得分3:=IF(MA10>REF(MA10,1) AND MA20>REF(MA20,1) AND MA50>REF(MA50,1),0.6,0); 
```
- 10日、20日、50日均线同时向上: +0.6分

#### 得分4: 价格相对于多空线位置
```tcl
得分4:=IF(C>知行多空线,0.3,0);
```
- 价格在多空线上方: +0.3分

#### 得分5: KDJ金叉状态
```tcl
得分5:=IF(NOT(J<K AND J<D),0.3,0); 
```
- 不是死叉状态(J不同时小于K和D): +0.3分

#### 得分6: 价格相对BBI位置
```tcl
得分6:=IF(C>BBI,0.4,0);
```
- 价格在BBI上方: +0.4分

#### 得分7: RSI相对枢轴线
```tcl
得分7:=IF(RSI1>枢轴线,0.5,0) + IF(RSI1>枢轴线-5,0.3,2); 
```
- RSI1>枢轴线: +0.5分
- RSI1>枢轴线-5: +0.3分 (接近但未超过)

#### 得分25: J值拐点
```tcl
得分25:=IF(J>REF(J,1) AND REF(J,1)<REF(J,2), 0.5, 0);
```
- J值今日上升且昨日下降(金叉): +0.5分

### 9.2 K线形态得分

#### 得分8: 上影线评估
```tcl
得分8:=IF(上影线<=K线长度/4,1.5,0) + IF(上影线>=K线长度/4,-1,0) + IF(上影线>=K线长度/2,-1,0);
```
- 上影线<=1/4: +1.5分 (强势形态)
- 上影线>=1/4: -1分
- 上影线>=1/2: -1分 (冲高回落)

#### 得分9: 上下影线同时短
```tcl
得分9:=IF(上影线<=K线长度/4 AND 下影线<=K线长度/4 AND C>O,1.2,0);
```
- 上下影线都短且收阳: +1.2分 (强势形态)

### 9.3 成交量得分

#### 得分10: 成交量活跃度
```tcl
得分10:=IF(VOL > VOL_MA60, 0.4, 0) + IF(VOL > 1.25 * VOL_MA60, 0.6, 0) + IF(VOL > 1.5 * VOL_MA60, 0.8, 0);
```
- 成交量>60日均量: +0.4分
- 成交量>1.25倍60日均量: +0.6分
- 成交量>1.5倍60日均量: +0.8分

#### 得分19: 倍量评估
```tcl
得分19:=IF(VOL>参考成交量*1.8,0.8,-2)+IF(VOL>参考成交量*2.5,0.3,0)+IF(VOL>参考成交量*3,0.3,0)+IF(VOL>参考成交量*3.5,0.3,0)+IF(VOL>参考成交量*4,0.3,0);
```
- 倍量(>1.8倍): +0.8分 (否则-2分)
- 2.5倍: +0.3分
- 3倍: +0.3分
- 3.5倍: +0.3分
- 4倍: +0.3分

#### 得分20/21: 关键K和暴力K
```tcl
得分20:=IF(关键K,1,0);
得分21:=IF(关键K AND 暴力K,0.5,0);
```
- 关键K: +1分
- 关键K+暴力K: +0.5分

### 9.4 趋势强度得分

#### 得分11: 创历史新高
```tcl
得分11:=IF(C=HHV(C,90),0.7,0) + IF(C=HHV(C,60),0.7,0) + IF(C=HHV(C,30),0.7,0);
```
- 90日新高: +0.7分
- 60日新高: +0.7分
- 30日新高: +0.7分

#### 得分14: 突破多空线
```tcl
得分14:=IF(REF(C,1)<REF(知行多空线,1) AND C>知行多空线,0.5,0) AND C>REF(H,1);
```
- 昨日在多空线下方，今日突破多空线且突破昨日最高价: +0.5分

#### 得分15: DIF与历史高点比较
```tcl
得分15:=IF(价格高点位置>0 AND 高点DIF < 前20高DIF, -0.3, 0) + IF(价格高点位置>0 AND 高点DIF < 前15高DIF, -0.3, 0);
```
- 当前位置DIF低于前20日DIF高点: -0.3分
- 当前位置DIF低于前15日DIF高点: -0.3分

### 9.5 阳线阴线对比得分

#### 得分12: 40日阳量阴量对比
```tcl
阳量40日总和:= SUM(IF(C>O,VOL,0),40); 
阴量40日总和:= SUM(IF(C<O,VOL,0),40); 
得分12:=(阳量40日总和 > 阴量40日总和) * 0.3 + (阳量40日总和 >= 1.25*阴量40日总和) * 0.4 + (阳量40日总和 >= 1.5*阴量40日总和) * 0.5;
```
- 阳量总和>阴量总和: +0.3分
- 阳量>=1.25倍阴量: +0.4分
- 阳量>=1.5倍阴量: +0.5分

#### 得分13: 20日单日最大阳量阴量对比
```tcl
最大阳量20日:=MAX(IF(C>O,VOL,0),20); 
最大阴量20日:=MAX(IF(C<O,VOL,0),20); 
得分13:=IF(最大阳量20日>最大阴量20日,0.5,0);
```
- 20日内最大阳量>最大阴量: +0.5分

### 9.6 成交量模式得分

#### 得分16/17: 历史天量后走势
```tcl
得分16:=(BARSLAST(V=HHV(V,60))>=1 AND REF(C,BARSLAST(V=HHV(V,60)))>REF(O,BARSLAST(V=HHV(V,60))))*0.3 +
		(BARSLAST(V=HHV(V,50))>=1 AND REF(C,BARSLAST(V=HHV(V,50)))>REF(O,BARSLAST(V=HHV(V,50))))*0.3 +
		(BARSLAST(V=HHV(V,40))>=1 AND REF(C,BARSLAST(V=HHV(V,40)))>REF(O,BARSLAST(V=HHV(V,40))))*0.3;
得分17:=(BARSLAST(V=HHV(V,30))>=1 AND REF(C,BARSLAST(V=HHV(V,30)))>REF(O,BARSLAST(V=HHV(V,30))))*0.3 +
    	(BARSLAST(V=HHV(V,20))>=1 AND REF(C,BARSLAST(V=HHV(V,20)))>REF(O,BARSLAST(V=HHV(V,20))))*0.3 +
    	(BARSLAST(V=HHV(V,10))>=1 AND REF(C,BARSLAST(V=HHV(V,10)))>REF(O,BARSLAST(V=HHV(V,10))))*0.3 +
		(BARSLAST(V=HHV(V,10))>=1 AND REF(C,BARSLAST(V=HHV(V,10)))<REF(O,BARSLAST(V=HHV(V,10)))) * (-2.5);
```

历史天量后股价收阳(继续上涨): 正向得分
历史天量后股价收阴(冲高回落): -2.5分

#### 得分18: 大盘配合
```tcl
得分18:=IF(上证收盘 > 上证昨高, 0.3, 0) + IF(上证收盘 > 上证30日高, 1, 0);
```
- 收盘>昨日高点: +0.3分
- 收盘>30日最高价: +1分

#### 得分23: 连续缩量且连续下跌
```tcl
得分23:=IF(V<REF(V,1)*1.1 AND REF(V,1)<REF(V,2)*1.1 AND REF(V,2)<REF(V,3)*1.1 AND REF(V,3)<REF(V,4)*1.1
		AND REF(C,1)<REF(O,1) AND REF(C,2)<REF(O,2) AND REF(C,3)<REF(O,3) AND REF(C,4)<REF(O,4),-1,0);
```
- 连续5日缩量且连续收阴: -1分 (弱势信号)

#### 得分24: 成交量创阶段新高
```tcl
得分24:=IF(VOL = HHV(VOL, 60),0.5,0) + IF(VOL = HHV(VOL, 50),0.5,0) + IF(VOL = HHV(VOL, 40),0.5,0) + IF(VOL = HHV(VOL, 30),0.5,0);
```
- 60日最高量: +0.5分
- 50日最高量: +0.5分
- 40日最高量: +0.5分
- 30日最高量: +0.5分

#### 得分27: 缩量评估
```tcl
得分27:=IF(V < HHV(V, 13), -0.3, 0) + IF(V < HHV(V, 12), -0.3, 0) + IF(V < HHV(V, 11), -0.3, 0);
```
- 成交量低于13日内最高量: -0.3分
- 成交量低于12日内最高量: -0.3分
- 成交量低于11日内最高量: -0.3分

### 9.7 负面信号得分

#### 得分26: 大长阴出现
```tcl
得分26:=IF(COUNT(大长阴 AND VOL > REF(VOL, 1) * 1.2 AND VOL > MA(VOL, 60) AND  VOL > (HHV(VOL, 60))/1.95, 15) >= 1, -0.4, 0) + 
		IF(COUNT(大长阴 AND VOL > REF(VOL, 1) * 1.4 AND VOL > MA(VOL, 60) AND  VOL > (HHV(VOL, 60))/1.95, 10) >= 1, -0.4, 0) +
```
