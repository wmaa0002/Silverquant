# 天宫B1策略通达信公式详解

```通达信
X:=10;                    { 买入门槛：B1总分 >= 10 才触发买入 }
M1:=14;M2:=28;M3:=57;M4:=114; { 均线周期参数 }
N1:=3;N2:=14;N3:=28;N4:=57;   { RSI周期参数 }
```

## 二、核心指标计算

### 2.1 知行趋势线

```通达信
知行短期趋势线:=EMA(EMA(C,10),10)
```
**解释**: 对收盘价进行10日EMA计算，再对结果进行10日EMA（双重平滑）

```通达信
知行多空线:=(MA(CLOSE,M1)+MA(CLOSE,M2)+MA(CLOSE,M3)+MA(CLOSE,M4))/4
        =(MA(C,14)+MA(C,28)+MA(C,57)+MA(C,114))/4
```
**解释**: 4条不同周期均线的平均值，作为多空分界线

### 2.2 KDJ指标

```通达信
RSV:=(C-LLV(L,9))/(HHV(H,9)-LLV(L,9))*100
K:=SMA(RSV,3,1)    { K = 2/3 * 前一K值 + 1/3 * RSV }
D:=SMA(K,3,1)     { D = 2/3 * 前一D值 + 1/3 * K }
J:=3*K-2*D
KDJ_J低 := J < 13
```
**解释**: 
- RSV: 未成熟随机值，收盘价在9日高低价区间位置
- K、D: 经过3日SMA平滑
- J: 3*K-2*D，超买超卖指标
- **KDJ_J低**: J值小于13时为低位的买入信号

### 2.3 MACD指标

```通达信
DIF:=EMA(C,12)-EMA(C,26)     { 12日EMA - 26日EMA }
DEA:=EMA(DIF,9)              { DIF的9日EMA }
MACD_多头 := DIF >= 0
```
**解释**: DIF大于等于0表示多头市场

### 2.4 其它技术指标

```通达信
涨幅:=(C-REF(C,1))/REF(C,1)*100          { 当日涨幅 }
振幅:=(H-L)/REF(C,1)*100                { 当日振幅 }
BBI:=(MA(C,3)+MA(C,6)+MA(C,12)+MA(C,24))/4  { 多空指标 }
前20日BBI:=REF(BBI,20)                    { 20天前的BBI值 }
```

```通达信
RSI1:=SMA(MAX(CLOSE-LC,0),N1,1)/SMA(ABS(CLOSE-LC),N1,1)*100
     =SMA(MAX(CLOSE-REF(C,1),0),3,1)/SMA(ABS(CLOSE-REF(C,1)),3,1)*100
RSI2,RSI3,RSI4: 周期分别为14、28、57的RSI
```

## 三、特殊形态定义

### 3.1 涨停相关

```通达信
涨停价:=IF(NAMELIKE('ST'),ZTPRICE(REF(C,1),0.05),
        IF(CODELIKE('300') OR CODELIKE('688') OR CODELIKE('301'),ZTPRICE(REF(C,1),0.2),
        ZTPRICE(REF(C,1),0.1)));
一字涨停:=(O=C)AND(C=H)AND(C=涨停价)
```
**解释**: 
- ST股：5%涨停
- 300/688/科创板：20%涨停
- 其它：10%涨停
- 一字涨停：开盘价=收盘价=最高价=涨停价

### 3.2 K线形态

```通达信
大长阳:=C>O AND 涨跌幅>波动率*1.5 AND 涨跌幅>2
大长阴:=C<O AND ABS(涨跌幅)>波动率*1.1 AND ABS(涨跌幅)>2
关键K:=CLOSE>REF(CLOSE,1) AND VOL>参考成交量*1.8 AND 大长阳 AND VOL>MA(VOL,40)
暴力K := C>REF(C,1) AND VOL>参考成交量*1.8 AND 涨跌幅>4 AND 上影线<=K线长度/4 AND VOL>MA(VOL,60)
```
**解释**:
- 大长阳：涨幅超过2%且超过波动率的1.5倍
- 大长阴：跌幅超过2%且超过波动率的1.1倍
- 关键K：放量长阳
- 暴力K：放量长阳且涨幅>4%

### 3.3 次高点判断

```通达信
次高点基础:=HHV(H,4)=HHV(H,60) AND H<>HHV(H,60) 
            AND (VOL>MA(VOL,5) OR VOL>MA(VOL,10)) 
            AND C<O 
            AND (((C/REF(C,10)-1)*100>10) OR ((C/REF(C,50)-1)*100>50))
次高点判断:=IF(次高点基础, IF(前3天最高点位置>=0, IF(VOL>=REF(VOL,前3天最高点位置)*0.80,1,0),0),0)
```
**解释**: 满足特定条件的次高点形态

## 四、B1分数计算条件（39个条件）

### 【条件组1】基础技术指标

| 条件代码 | 得分 | 条件说明 |
|---------|------|----------|
| 条件1 | +0.6 / 0 | DIF >= 0（多头） |
| 条件2 | +0.3 / 0 | 收盘价 > 60日均线 |
| 条件3 | +1.5 / -3 | 涨幅在 -2% ~ 1.8% 之间 |
| 条件4 | +0.5 / -1 | 振幅 < 7% |
| 条件5 | +0.8 / 0 | 振幅 < 4% |
| 条件6 | +1.5 max | RSI1 < 20 得0.8分，< 23 得0.7分 |

**解读**: 条件1-6主要考察MACD方向、价格位置、波动性、RSI超卖

### 【条件组2】成交量地量信号

| 条件代码 | 得分逻辑 |
|---------|----------|
| 条件8 | 当成交量为30/26/24/22/20日最低时，各得0.3分 |
| 条件9 | 当成交量为18/16/14/12/10日最低时，各得0.3分 |
| 条件10 | 当成交量为20/19/18日最低时，得0.5分 |

**解读**: 连续地量信号是重要的买入信号

### 【条件组3】量价配合

```通达信
条件11: 倍量柱+阳线
  (BARSLAST(V=HHV(V,60))>=1 AND REF(C,BARSLAST(V=HHV(V,60)))>REF(O,...)

条件12: 倍量柱+阴线 (负分)
条件13: 近期倍量柱+阴线 (负分)
```
**解读**: 放量且阳线上涨是积极信号，放量下跌是风险信号

### 【条件组4】涨跌量对比

```通达信
条件14: (SUM(IF(C>O,V,-V),50)>0)*0.4 + ...
条件15: (SUM(IF(C>O,V,0),30) > 1.25*SUM(IF(C<O,V,0),30))*0.4 + ...
```
**解读**: 统计50/40/30/20日的涨跌量和，净流入为正加分

### 【条件组6】DIF背离

```通达信
条件22: IF(价格高点位置>0 AND 高点DIF<前20高DIF, -0.5, 0) +
```

## 五、B1分数计算总结

### 5.1 计算公式

```通达信
B1得分 := 条件1+条件2+...+条件39

B1总分 := IF(KDJ_J低 AND B1得分>0, B1得分, 0)
```

**解读**: 
- B1总分 = B1得分（仅当J<13且B1得分>0时，否则为0）
- 体现了"KDJ低位 + 技术面综合得分"的双重筛选

### 5.2 买入条件

```通达信
选股 := 非ST股 AND KDJ_J低 AND MACD_多头 AND 知行短期趋势线 > 知行多空线 AND B1总分 >= X(X=10)
```

**完整买入条件**:
1. ✅ 非ST股票
2. ✅ KDJ的J值 < 13（超卖状态）
3. ✅ DIF >= 0（多头趋势）
4. ✅ 知行短期趋势线 > 知行多空线（上升趋势）
5. ✅ B1总分 >= 10（综合技术面合格）

### 5.3 分数分布特点

| 分数区间 | 市场含义 | 建议 |
|---------|----------|------|
| 0-5分 | 技术面较弱 | 观望 |
| 5-10分 | 技术面一般 | 谨慎买入 |
| 10-15分 | 技术面良好 | 建议买入 |
| 15-20分 | 技术面优秀 | 强烈买入 |
| 20+分 | 极强信号 | 重点关注 |

## 七、策略核心逻辑

天宫B1策略是一个**趋势回调买入**策略：

1. **择时**: 在J值超卖时买入（抄底）
2. **趋势确认**: 依靠DIF>=0、知行线多头排列确认趋势向上
3. **综合评分**: 通过39个技术指标综合打分，确保技术面配合
4. **风险过滤**: 排除ST股、排除大跌后的弱势股

**核心理念**: "买在回调点，卖在起涨时"
