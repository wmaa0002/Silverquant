### 4. 暴力K信号

**策略文件：** `天宫暴力K策略V1.py`

**暴力K的7个条件：**
1. C > REF(C, 1) - 收盘价高于昨日
2. VOL > 参考成交量 * 1.8 - 成交量放大1.8倍以上
3. 涨跌幅 > 4 - 涨幅大于4%
4. 上影线 <= K线长度 / 3.5 - 上影线限制
5. 前9日涨幅 < 4 - 前期涨幅限制
6. 前4日涨幅 < 4 - 近期涨幅限制
7. VOL > MA(VOL, 60) - 成交量大于60日均量

**Python实现代码：**

### 5. 地量信号

**策略文件：** `天宫地量策略V1.py`

**地量等级计算逻辑：**
- 地量1: V*0.9 <= LLV(V, 10~20) → 得分10~20
- 地量2: V*0.9 <= LLV(V, 21~30) → 得分21~30
- 地量3: V*0.9 <= LLV(V, 31~40) → 得分31~40
- 地量4: V*0.9 <= LLV(V, 41~50) → 得分41~50
- 综合地量 = MAX(地量1, 地量2, 地量3, 地量4)

**Python实现代码：**
```python
def calculate_dl_score(self, close, high, low, open_price, close_arr, high_arr, low_arr, volume_arr, volume, prev_close, debug=False):
    
    # 排除涨停板/跌停板/一字涨停/一字跌停
    if 涨停板 or 跌停板 or 一字涨停 or 一字跌停:
        return 0
    
    # 地量1: V*0.9 <= LLV(V, 10~20)
    地量1 = 0
    for days in range(10, 21):
        if len(volume_arr) >= days:
            llv_vol = np.min(volume_arr[-days:])
            if volume * 0.9 <= llv_vol:
                地量1 = days
                break
    
    # 地量2、3、4 同理...
    
    # 综合地量: 优先级 地量4 > 地量3 > 地量2 > 地量1
    地量等级 = 0
    if 地量4 >= 41: 地量等级 = 地量4
    elif 地量3 >= 31: 地量等级 = 地量3
    elif 地量2 >= 21: 地量等级 = 地量2
    elif 地量1 >= 10: 地量等级 = 地量1
    
    return 地量等级
```

**买入条件：**
```python
buy_condition = (not self.st_days and dl_score >= self.p.dl_threshold)
```

### 6. SCB综合信号（沙尘暴策略）

**策略文件：** `天宫沙尘暴策略V1.py`

**SCB综合信号组成：**
1. **地量基础条件**（过去5日内任一天满足）：
   - 地量 >= X (默认30)

   - VOL > 0
   - 前60日非阴
   - 60日内有关键K或暴力K
   - 80日内有异动（VOL > MA(VOL,60) * 2 AND C>O）

2. **暴力K信号**（当日满足）

3. **多头发散**：
   - 知行多空线 > REF(知行多空线, 60)

**Python实现代码：**
