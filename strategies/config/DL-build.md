## 四、建议补充内容

### 4.1 建议新增指标（高优先级）

1. **涨停板/跌停板标志**
   ```
   涨停板 = C >= 涨停价 AND C > O
   跌停板 = C <= 跌停价 AND C < O
   ```

2. **地量指标系列**
   ```
   地量1 = V*0.9 <= LLV(V, 10~20)
   地量2 = V*0.9 <= LLV(V, 21~30)
   地量3 = V*0.9 <= LLV(V, 31~40)
   地量4 = V*0.9 <= LLV(V, 41~50)
   地量 = 取满足条件的地量等级
   ```

3. **非ST股判断**
   ```
   非ST股 = NOT(NAMELIKE('ST')) AND NOT(NAMELIKE('*ST'))
   ```

4. **TR（真实波幅）**
   ```
   TR = MAX(High-Low, ABS(High-REF(Close,1)), ABS(Low-REF(Close,1)))
   ```

### 4.2 建议补充指标（中优先级）

1. **倍量柱完整定义**
2. **异动80定义**
3. **前60日非阴判断**

## 五、DL选股公式完整条件

```
选股条件:
  地量 >= X
  AND 非ST股
  AND FINANCE(42) > 100  {流通市值}
  AND VOL > 0
  AND 前60日非阴
  AND (COUNT(关键K,60) >= 1 OR COUNT(暴力K,60) >= 1)
  AND 异动80
```
