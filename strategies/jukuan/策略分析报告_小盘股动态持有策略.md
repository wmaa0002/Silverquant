## 三、选股模块详解

### 3.1 选股流程

```
filter_stocks() → 财务筛选 → 市值排序 → 价格过滤 → 最终持仓
```

### 3.2 股票过滤规则 (filter_stocks)

| 过滤条件 | 代码逻辑 | 说明 |
|----------|----------|------|
| 停牌过滤 | `current_data[stock].paused` | 跳过停牌股票 |
| ST过滤 | `current_data[stock].is_st` | 跳过ST股票 |
| 退市过滤 | `'退' in name` | 跳过退市股票 |
| 板块过滤 | `startswith('30','68','8','4')` | 过滤科创板/创业板/北交所 |
| 涨停过滤 | `last_price >= high_limit` | 不买已涨停 |
| 跌停过滤 | `last_price <= low_limit` | 不买已跌停 |
| 次新股过滤 | `start_date - today < 375天` | 上市不足375天不买 |

### 3.3 财务筛选条件

```python
valuation.code.in_(initial_list)                    # 深证成指成分股
valuation.market_cap.between(10, 100)               # 总市值10-100亿
income.np_parent_company_owners > 0                # 归属于母公司净利润>0
income.net_profit > 0                              # 净利润>0
income.operating_revenue > 1e8                    # 营业收入>1亿元
```

### 3.4 排序与选股

- **排序方式**: 按总市值升序（从小到大）
- **候选数量**: `stock_num * 3` = 12只（默认4*3）
- **价格过滤**: 持仓股票或价格<=50元的股票

## 四、动态调仓模块 (adjust_stock_num)

### 4.1 调仓逻辑

基于深证成指399101的MA10与收盘价差值，动态调整持仓数量：

| 差值范围 | 持仓数量 |
|----------|----------|
| diff >= 500 | 3只 |
| 200 <= diff < 500 | 3只 |
| -200 <= diff < 200 | 4只 |
| -500 <= diff < -200 | 5只 |
| diff < -500 | 6只 |

### 4.2 调仓规则

- **卖出条件**: 不在目标持仓中 且 不是昨日涨停股
- **买入条件**: 在目标持仓中 且 不在当前持仓中
- **资金分配**: 等金额分配到各只股票

## 五、止损止盈模块

### 5.1 个股止盈 (stoploss_strategy = 1 或 3)

```python
if price >= avg_cost * 2:    # 盈利100%
    order_target_value(stock, 0)  # 全部卖出
```

### 5.2 个股止损 (stoploss_strategy = 1 或 3)

```python
if price < avg_cost * (1 - 0.09):  # 亏损9%
    order_target_value(stock, 0)    # 止损卖出
```

### 5.3 市场止损 (stoploss_strategy = 2 或 3)

```python
# 计算指数成分股平均涨跌幅
down_ratio = (1 - close / open).mean()

if down_ratio >= 0.05:  # 平均跌幅>=5%
    # 全部清仓
    for stock in current_positions:
        order_target_value(stock, 0)
```

## 十、策略特点分析

### 10.1 优势

1. **小盘股策略**: 专注10-100亿小盘股，弹性大
2. **动态持仓**: 根据市场趋势调整持仓数量，牛市多持股
3. **严格过滤**: 多重过滤排除问题股票
4. **止损机制**: 多层止损保护本金
5. **涨停持有**: 不轻易卖出强势股

### 10.2 风险

1. **小盘股波动大**: 10亿市值股票可能暴涨暴跌
2. **频繁交易**: 每周调仓，交易成本较高
3. **市场止损敏感**: 大盘急跌可能卖在低点
4. **次新股风险**: 375天过滤期可能错过新股机会
